قوانين الفيزياء - الصف الثاني عشر (متقدم)

الوحدة 1: الحركة في بعد واحد

  • السرعة المتوسطة v = Δx / Δt
  • التسارع a = Δv / Δt
  • السرعة النهائية v_f = v_i + a·t
  • الإزاحة Δx = v_i·t + 0.5·a·t²
  • السرعة من دون زمن v_f² = v_i² + 2a·Δx

الوحدة 2: القوى وقوانين نيوتن

  • القوة المحصلة F = m·a
  • الوزن W = m·g
  • الاحتكاك الحركي f_k = μ_k · N
  • الاحتكاك الساكن f_s ≤ μ_s · N
  • قانون نيوتن الثالث F₁ = -F₂

الوحدة 3: الطاقة والعمل

  • الشغل W = F·d·cos(θ)
  • الطاقة الحركية KE = 0.5·m·v²
  • الطاقة الكامنة PE = m·g·h
  • الطاقة الميكانيكية الكلية E = KE + PE
  • قانون حفظ الطاقة E_{initial} = E_{final}

الوحدة 4: الزخم والتصادمات

  • الزخم p = m·v
  • قانون حفظ الزخم m₁·v₁ + m₂·v₂ = m₁·v₁' + m₂·v₂'
  • القوة والزخم F = Δp / Δt

الوحدة 5: الحركة الدورانية

  • السرعة الزاوية ω = θ / t
  • التسارع الزاوي α = Δω / Δt
  • الإزاحة الزاوية θ = ω_i·t + 0.5·α·t²
  • العزم τ = r·F·sin(θ)
  • العزم المحصّل τ = I·α
  • طاقة الحركة الدورانية KE = 0.5·I·ω²

الوحدة 6: الجاذبية

  • قوة الجاذبية (نيوتن) F = G·(m₁·m₂) / r²
  • تسارع الجاذبية g = G·(M) / r²

الوحدة 7: الذبذبات والموجات

  • الزمن الدوري T = 1 / f
  • السرعة الموجية v = λ·f
  • طاقة النابض PE = 0.5·k·x²
  • حركة النابض F = -k·x

الوحدة 8: الكهرباء (مقدمة فقط)

  • قانون أوم V = I·R
  • القدرة الكهربائية P = V·I
  • الطاقة الكهربائية E = P·t
